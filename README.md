# Temperature Converter

A simple, modern temperature converter that supports **Celsius (°C)**, **Fahrenheit (°F)**, and **Kelvin (K)**.

## Features

- **3-unit conversion**: °C ⇄ °F ⇄ K
- **Swap units** button
- Shows the **formula** used for the conversion
- Keyboard support: press **Enter** to convert

## How to Run

- Option 1: Open `index.html` in your browser
- Option 2 (recommended): Use VS Code **Live Server**

## Project Structure

- `index.html` — UI markup
- `style.css` — styling
- `script.js` — conversion logic

## Notes

- Kelvin uses the standard offset: **K = °C + 273.15**
